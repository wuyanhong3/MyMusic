# MyMusic
#音乐播放器
  实现的功能：后台播放，收藏喜爱的音乐到数据库，可以滑动专辑封面切歌，通知栏显示当前播放歌曲和控制歌曲播放，有单曲循环，列表循环，随机播放和顺序播放
     四种播放播放模式;
##app用到的主要知识点：
     service:后台播放音乐；<br>
     BroadCastReciver:通知栏控制音乐；<br>
     ContentProvider：通过uri读取手机媒体数据库；<br>
     Sqlite:保存收藏的音乐；<br>
     Fragment+viewPager：滑动专辑封面切歌；<br>
